Hieroglyphic Transliterator

Convert Gardiner sign codes (e.g., A1, G43) or MdC alphabet (e.g., m, n, i) into copyable Unicode Egyptian Hieroglyphs. Built for the Egyptology Research Archive.

This tool is built with a few powerful features specifically for Egyptology:

  • Transliteration Mapping: It recognizes academic transliteration symbols (e.g., ꜣ, ḫ, š, ḏ). Typing words like nfr or bꜣ will automatically pull the correct sign sequences.
  • Intelligent Parsing: It understands standard Gardiner codes (e.g., M17, G43) regardless of capitalization, as well as Manuel de Codage (MdC) uniliterals (e.g., typing i w t translates instantly to 𓇋𓅱𓏏).
  • Formatting Controls: It supports Unicode grouping controls (: for vertical stacking and * for horizontal grouping) so you can accurately represent sign positioning.
  • Embedded Dictionary: It includes a dense dictionary of hundreds of the most common Gardiner signs (including all uniliterals, numbers, and common determinatives) directly in the file so it works instantly.
