Hieroglyphic Transliterator

Convert Gardiner sign codes (e.g., A1, G43) or MdC alphabet (e.g., m, n, i) into copyable Unicode Egyptian Hieroglyphs. Built for the Egyptology Research Archive.

This tool is built with a few powerful features specifically for Egyptology:

  • Intelligent Parsing: It understands standard Gardiner codes (e.g., M17, G43) regardless of capitalization, as well as Manuel de Codage (MdC) single-letter uniliterals (e.g., typing i w t translates instantly to 𓇋𓅱𓏏).
  • Formatting Controls: It supports Unicode grouping controls (: for vertical stacking and * for horizontal grouping) so you can accurately represent sign positioning.
  • Embedded Dictionary: It includes a dense dictionary of hundreds of the most common Gardiner signs (including all uniliterals, numbers, and common determinatives) directly in the file so it works instantly without needing external APIs.
